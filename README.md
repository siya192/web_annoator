Description

Web Annotator is a Chrome extension that allows users to highlight text on any webpage and attach notes to the highlighted content. The annotations are stored locally in the browser so that users can revisit their highlights and notes across browser sessions.

Features

Highlight Content
Highlight important text on any webpage.

Add Notes
Click on a highlighted text to add or edit personal notes.

Persistence
Highlights and notes are saved using browser storage and persist across browser sessions for the same webpage.

Works on Any Website
Can be used on articles, blogs, documentation, and educational websites.

Tech Stack

JavaScript

HTML

CSS

Chrome Extensions API

Browser Local Storage

How It Works

User selects text on a webpage.

The selected text is highlighted.

Clicking on the highlighted text opens a prompt to add notes.

Annotations are saved in browser storage.

When the user revisits the page, saved notes can be accessed again.

Installation Steps

Clone or download this repository.

Open Google Chrome and go to chrome://extensions.

Enable Developer Mode.

Click on Load unpacked.

Select the project folder.

The Web Annotator extension will be added to Chrome.

Usage

Select any text on a webpage to highlight it.

Click on the highlighted text to add or edit notes.

Reloading the page does not remove saved notes.

Limitations

Highlight restoration works best on static content.

Dynamic websites may not fully restore highlights due to DOM changes.

Annotations are stored locally and are not synced across devices.
